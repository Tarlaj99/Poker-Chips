# Poker-Chips
Welcome to Poker Chips, a web application designed to replace physical poker chips during live poker games. 

# WARNING: THIS IS NOT AN ONLINE POKER APP
The purpose of this application is to replace physical poker chips during live poker games.

# SETUP
Before starting the server, run the following commands.:

npm install
node db_setup.js

# START:
in order to start a web server you need to type in the terminal:

node index.js

# NOTE: The app is designed for to be used by multiple user groups. If you're testing it i suggest you make a new incognito tab in your browser in order to have a few sessions at the same time.
